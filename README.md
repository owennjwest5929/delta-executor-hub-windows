## About Delta Executor

Delta Executor is a Windows desktop Roblox script executor made for users who want a clean, local way to run Lua scripts. It brings together script execution, a script hub, and a lightweight WPF interface so the experience stays quick and straightforward on Windows 10 and Windows 11.

This release is built around practical day-to-day use, including auto-update support, batch execution, and a built-in debugger.
